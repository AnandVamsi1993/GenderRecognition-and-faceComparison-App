FaceRecognition Application


Gender Classification: This app is driven by data and machinelearning model. As soon as user uploads the image, the tool will convert this image to GrayScale, use Haarcascade functionality to recognize faces, save the faces in a directory, preprocess the image for efficient computing and predict gender based on a model built on Support Vector Machines. All these functionalities are then dumped on the flask application.
